Class VRM

Represents a VRM model. It has so many feature to deal with your VRM models!

example

Most basic use of VRM

const scene = new THREE.Scene();

new THREE.GLTFLoader().load( 'models/shino.vrm', ( gltf ) => {

  THREE.VRM.from( gltf ).then( ( vrm ) => {

    scene.add( vrm.scene );

  } );

} );
